哈利波特的神奇日记本

小哈利波特在汤姆里德尔的笔记上写字，停笔等一会儿，刚写下去的字会慢慢消失。一行新的手写体从纸面上浮出来，像是另一头的汤姆里德尔给哈利的回信。

使用说明
  1. 需要拿到最新的密钥钥匙才可以体验（向管理员索取；后台 /admin 可生成）。
  2. 进入日记本后，点右上角 ⚙，填入自定义提供商 Provider。
  3. 填 BASE_URL —— OpenAI 兼容接口地址。
  4. 填 MODEL —— 须是视觉模型（能看懂你写在纸上的字）。常用推荐：
    • meta/llama-3.2-11b-vision-instruct ← 小，便宜，够用，推荐先试
    • meta/llama-3.2-90b-vision-instruct ← 大，慢，效果好
    • microsoft/phi-3-vision-128k-instruct ← 小，视觉也能用
    • nvidia/llama-3.1-nemotron-nano-vl-8b-v1 ← 带名称里 -vl- 的是视觉模型
    • nvidia/nemotron-nano-12b-v2-vl ← n 家中量视觉
    • nvidia/llama-3.2-nemoretriever-1b-vlm-embed-v1 ← 带 -vlm
  5. 保存配置后停笔约 3 秒，墨迹被日记"喝掉"，回信会以手写体浮现。

判别小窍门：模型名里带 -vl- / -vlm / -vision 的就是视觉模型；只有文字模型看不懂你写的内容。
